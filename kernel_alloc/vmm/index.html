Module vmm

Minimal Virtual Memory Manager (VMM) for the kernel.

This VMM provides basic map/unmap/query operations for a single address space. It uses the AddressSpace abstraction from kernel-vmem, and FrameAlloc/PhysMapper implementations from kernel-alloc.

§Example

ⓘ
use kernel_alloc::{frame_alloc::BitmapFrameAlloc, phys_mapper::HhdmPhysMapper, vmm::Vmm};
let mut pmm = BitmapFrameAlloc::new();
let mapper = HhdmPhysMapper;
let mut vmm = Vmm::new(&mapper, &mut pmm);
// Map, unmap, query...

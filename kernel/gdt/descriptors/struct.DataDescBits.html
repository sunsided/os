#[repr(transparent)]
pub struct DataDescBits(u64);
Expand description

Bit layout of a data/stack segment descriptor.

Invariants enforced by from_data_dpl:

  • typ = 0b0010 (read/write data),
  • s = 1, l = 0, p = 1,
  • db is set to 0 here (commonly set to 1 for 32-bit data, but DB has no meaning for 64-bit data segments used as DS/ES/SS in long mode).

Expand description

Descriptor Privilege Level (lives in the descriptor).

Do not confuse with:

  • CPL (Current Privilege Level) โ taken from CS of the running code
  • RPL (Requested Privilege Level) โ low 2 bits in a segment selector

Typical checks the CPU enforces include:

  • Data segments: max(CPL, RPL) โค DPL
  • Non-conforming code: CPL == DPL
  • Conforming code: CPL โค DPL
  • SS load (long mode): CPL == RPL == DPL

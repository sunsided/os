#[repr(transparent)]
pub struct PageFaultError(u64);
Expand description

Page-fault error code layout (x86-64).

Each bit describes the condition that caused the page fault. Reference: Intel SDM Vol. 3A, §6.15.1 “Page-Fault Exception (#PF)”.

#[repr(transparent)]
pub struct Cr4(u64);
Expand description

CR4 — Control Register 4 (x86-64).

Controls paging, extended instruction state management, and various protection features (UMIP, SMEP/SMAP, PKE, …).

Only the low bits are architecturally defined; the rest are reserved.

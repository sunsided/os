pub struct Cr3(u64);
Expand description

CR3 — Page-Map Level-4 Base Register (IA-32e, PCID disabled).

Holds the physical base address of the PML4 table and cache-control flags for PML4 walks. Assumes standard 4 KiB alignment and no PCID (CR4.PCIDE = 0).

Struct Cr0 

Source 
#[repr(transparent)]
pub struct Cr0(u64);
Expand description

Architectural model of CR0 in 64-bit mode.

Exposes all architecturally defined control bits as booleans and keeps all reserved bits forced to 0. Upper 32 bits are also modeled as reserved.

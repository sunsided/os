Expand description

Hardware Page Size (PS) bit position shared across levels (bit 7).

  • In non-leaf entries: PS must be 0.
  • In large leaf entries (L3 1 GiB / L2 2 MiB): PS must be 1.
  • In L1 4 KiB PTEs: bit 7 is PAT (not PS).
